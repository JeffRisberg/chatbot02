# chatbot02

Technologies used:

- Node v14.19.1
- React 18.2
- Redux
- ES6
- Webpack
- Axios

## How to start

Run:
```
npm install
npm run webpack-watch
```

Then in a separate window, start the backend as:
```
node server
```

Check your http://localhost:3000/ or  `open http://localhost:3000/`
